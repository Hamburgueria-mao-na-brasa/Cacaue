Primeiro acesso

Como salvar para todos os clientes

Use este passo a passo quando entrar no painel pela primeira vez ou quando algo não aparecer em outro celular.

  1. Entre com e-mail e senha do Supabase.O painel só salva no servidor quando aparecer a mensagem de admin conectado.
  2. Cadastre ou edite o produto.Escolha categoria, subcategoria, preço, foto e disponibilidade.
  3. Toque em salvar e espere a confirmação.Procure a mensagem “salvo com sucesso no servidor”. Se aparecer erro, a alteração não foi enviada para outros celulares.
  4. Teste em outro aparelho.Abra o site novamente ou limpe o cache para confirmar que o servidor carregou a alteração.
